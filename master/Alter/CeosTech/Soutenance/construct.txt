Indexes :


logiciel SAAS : Un logiciel SAAS (software as a service) veut signifier un logiciel applicatif hébergé et exploité en dehors de l'organisation ou de l'entreprise par un tiers. L'outil est ainsi accessible à la demande via un accès Internet.

Application Native : Une application native veut signifier une application qui a été développé en une fois mais auquel son code a été conçu et converti pour fonctionner sur différents système. Dans notre cas, React Native nous permet de développer en une fois un code qui fournira un même rendu sur Android que sur IOS.

KPIs : Un KPI (Key Performance Indicators) est une fonctionnalité concentre tous les indicateurs clés de performance d'une entité précise sous la forme de données et graphiques. Le tableau de bord d'une voiture par exemple est ainsi un regroupement de KPIs

Fiche technique : Une fiche technique est fiche qui détaille la réalisation d'un plat servit en restauration, il s'agit autrement dit d'une recette de cuisine.

front-end : Le front-end concerne la partie d'une application, elle est chargée à la mise en place des éléments graphiques qui permettront à l’utilisateur final de se repérer sur l’interface ainsi qu'à la prendre facilement en mains. Le front va également prendre en compte le développement des interactions, des animations, du responsive design, de l'expérience et de l'interface utilisateur (UX/UI)

back-end : Le back-end concerne l'autre partie d'une application avec le front-end et est porteuse de ça valeur ajoutée. Le back-end est chargée de construire, de développer et de mettre en interaction les éléments essentiels au fonctionnement de l’application comme notamment la base de données et le serveur d'hébergement.

API : Une API (Application Programming Interface) est un moyen informatique qui permet à des applications de communiquer entre elles et de s'échanger mutuellement des services ou des données.

requête API : Une requête d'API est une methode servant l'échange de données entre les applications, il en existe de plusieurs sortes comme le GET et le POST étant les plus connus.

IDE : Un IDE (Integrated Development Environment) est logiciel qui regroupe un ensemble d’outils de développements dédiés aux programmeurs afin qu’ils puissent optimiser leur temps de travail et améliorer leur productivité. Autrement dit, l’IDE facilite la mise en œuvre de projets tels que le développement de logiciels ou d’applications.

CLI : un CLI est une interface homme-machine dans laquelle la communication entre l'utilisateur et l'ordinateur s'effectue en mode texte via donc l'utilistion de différentes commandes.
